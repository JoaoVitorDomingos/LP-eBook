# Contexto do Projeto

## Visão Geral

Este projeto consiste no desenvolvimento de uma **Landing Page de Alta Conversão** para divulgação e captura de leads de um **e-book de receitas saudáveis**.

O objetivo não é apenas apresentar informações sobre o produto, mas conduzir o usuário por uma jornada de convencimento, utilizando princípios de UX, UI Design e Copywriting para aumentar a taxa de conversão.

Este projeto faz parte do meu portfólio pessoal e está sendo reconstruído do zero para atingir um padrão visual e técnico próximo ao encontrado em produtos comerciais modernos.

---

# Objetivos

## Objetivo Principal

Converter visitantes em leads por meio do preenchimento de um formulário integrado ao Mailchimp.

## Objetivos Secundários

* Apresentar o e-book de forma clara e atrativa.
* Demonstrar os benefícios das receitas saudáveis.
* Gerar confiança por meio de provas sociais e argumentos de valor.
* Incentivar o usuário a realizar apenas uma ação: baixar o e-book.

---

# Público-Alvo

Pessoas interessadas em alimentação saudável, incluindo:

* Pessoas que desejam melhorar a alimentação.
* Quem busca receitas simples e práticas.
* Pessoas com rotina corrida.
* Iniciantes na alimentação saudável.
* Pessoas que desejam emagrecer ou melhorar seus hábitos alimentares.

Faixa etária predominante:

* 20 a 45 anos.

---

# Proposta de Valor

Oferecer gratuitamente um e-book contendo receitas saudáveis, práticas e saborosas, incentivando hábitos alimentares melhores de forma simples.

---

# Objetivo da Landing Page

A landing page possui apenas um objetivo:

**Convencer o usuário a preencher o formulário para receber o e-book.**

Qualquer elemento visual ou textual deve contribuir para esse objetivo.

---

# Tom da Comunicação

A comunicação deve transmitir:

* confiança;
* proximidade;
* profissionalismo;
* clareza;
* motivação.

Evitar linguagem excessivamente técnica ou informal.

O texto deve ser simples, objetivo e fácil de ler.

---

# Identidade Visual

A identidade visual deve transmitir:

* saúde;
* leveza;
* energia;
* qualidade;
* modernidade;
* confiança.

O projeto deve evitar aparência de blog antigo ou site institucional.

O objetivo é parecer uma landing page moderna de um produto digital.

---

# Estilo de Design

O design deverá seguir uma abordagem:

* minimalista;
* clean;
* premium;
* moderna;
* responsiva;
* focada em conversão.

Priorizar:

* bastante espaço em branco;
* tipografia bem definida;
* componentes consistentes;
* cards modernos;
* bordas suaves;
* sombras discretas;
* animações sutis.

---

# Referências Visuais

O projeto busca inspiração em empresas e produtos conhecidos pelo alto padrão de design, como:

* Stripe
* Linear
* Framer
* Vercel
* Notion
* Raycast

Essas referências servem apenas como inspiração para qualidade visual, organização e experiência do usuário, não devendo ser copiadas.

---

# Estrutura da Landing Page

A estrutura principal deverá seguir a seguinte ordem:

1. Hero Section
2. Problema
3. Benefícios
4. Conteúdo do e-book
5. Provas sociais
6. Chamada para ação (CTA)
7. Rodapé

Essa ordem deve ser mantida durante futuras alterações, salvo necessidade específica do projeto.

---

# Tecnologias

O projeto será desenvolvido utilizando:

* HTML5
* CSS3
* JavaScript (quando necessário)

Princípios adotados:

* Mobile First
* HTML Semântico
* CSS Modular
* Responsividade
* Acessibilidade
* Código limpo
* Boas práticas de SEO

---

# Diretrizes para Desenvolvimento

Durante a implementação, todas as alterações devem respeitar as documentações presentes na pasta `docs`.

Em caso de conflito entre decisões de implementação e a documentação, a documentação deve prevalecer.

Novos componentes, padrões visuais ou decisões de design devem ser registrados antes de serem reutilizados no projeto, garantindo consistência ao longo do desenvolvimento.

---

# Escopo

Este projeto não tem como objetivo criar um sistema completo de vendas.

O foco é exclusivamente a construção de uma Landing Page moderna, responsiva e de alta conversão para apresentação e captura de leads de um e-book de receitas saudáveis.
