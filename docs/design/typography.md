# Tipografia

| Papel        | Fonte             | Peso    | Uso                                                                             |
| ------------ | ----------------- | ------- | ------------------------------------------------------------------------------- |
| Display Font | **Space Grotesk** | 700     | Hero, H1, grandes títulos e números de impacto.                                 |
| UI Font      | **Manrope**       | 500-600 | H2, H3, botões, badges, cards, CTAs, estatísticas, labels e textos em destaque. |
| Body Font    | **Inter**         | 400–500 | Parágrafos, listas, formulários, FAQ e descrições.                              |
